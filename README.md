## Overview

Implementation of foundational algorithms for DNA sequence analysis as part of the Johns Hopkins Bioinformatics specialization. Covers FASTA/FASTQ file parsing, naive exact matching, naive approximate matching with up to 2 mismatches, and read quality analysis using Phred scores.

## Implemented Algorithms

| Algorithm | Description |
|---|---|
| FASTA/FASTQ Parser | Reads and parses standard genomic file formats |
| Naive Exact Matching | Brute-force pattern matching against a reference genome |
| Naive Approximate Matching | Pattern matching allowing up to k mismatches |
| Quality Score Analysis | Phred score extraction and per-position quality profiling |

## Key Concepts

- Pattern matching complexity and alignment fundamentals
- Read quality filtering and sequencing error profiles
- File format standards in genomic data science
